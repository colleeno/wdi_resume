weather planner

 

Single page web application created with HTML, CSS, Javascript, jQuery, Ajax.
Enter any city to view weather + suggested weather appropriate activities. Click an activity to view local results. Apixu API used for weather and Google Places Javascript Library used for location autocomplete and place results.

Full stack Rails application. Created with Ruby on Rails, HTML, CSS, jQuery, Masonry layout.
View popular tv and movie quotes, guess their author, and star your favorites. Logged in users have full crud functionality.

Full stack Rails application. Created with Ruby on Rails, HTML, and CSS.
